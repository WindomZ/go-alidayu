# go-alidayu

## Overview

* 基于最新阿里大鱼API
* 提供了简单轻量化的并发队列
* 方便自行封装
* 支持回调跟踪

# Getting Started

## Installation

```
go get -u github.com/WindomZ/go-alidayu
```

## Documentation 

可以先基于 *_test.go 进行了解
(需要在`tests/init.go`中配置正确的API调用参数)
